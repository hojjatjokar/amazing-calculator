## How to use

1. Clone the project
2. `npm i`

## How to contribute

1. Fork the repo
2. Clone your fork
3. Add upstream remote
4. Create a branch
5. Add your commits (before commit read [conventional commits guide](https://www.conventionalcommits.org/en/v1.0.0/))
6. Push your branch
7. Create a PR
